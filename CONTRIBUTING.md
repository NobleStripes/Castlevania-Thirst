# 🧛‍♂️ Contributing to the Crypt

We welcome contributions, but only if you follow the **Ancient Laws of the Night**.

## 📜 The Rules

1. **No Holy Water**: Any code containing `holy-water` or `cross-shaped-logic` will be immediately incinerated.
2. **Respect the Drip**: If you're adding a new character, their CSS must be as sharp as Alucard's sword.
3. **Maternal Supremacy**: Do not, under any circumstances, disrespect the MILF Council (Lisa, Tera, Julia). Your PR will be closed and you will be sent to the Infinite Corridor.
4. **Beer for Trevor**: Every 100 lines of code must be accompanied by a virtual pint for Trevor Belmont.
5. **Dracula's Approval**: All major architectural changes must be approved by the King of Vampires. If he's busy ending the world, wait your turn.

## 🛠️ How to Contribute

1. **Fork the Crypt**.
2. **Create a Branch** (e.g., `feature/more-thirst`).
3. **Commit your changes** with a message that sounds like a gothic poem.
4. **Open a Pull Request** and pray to the Speaker elders.
