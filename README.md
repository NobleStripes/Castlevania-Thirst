## 🚀 Getting Started

1. **Clone the Crypt**:
   ```bash
   git clone https://github.com/your-username/castlevania-thirst-counter.git
   ```
2. **Summon Dependencies**:
   ```bash
   npm install
   ```
3. **Ignite the Forge**:
   ```bash
   npm run dev
   ```
4. **Open your browser** to `http://localhost:3000` and prepare to witness the thirst.
